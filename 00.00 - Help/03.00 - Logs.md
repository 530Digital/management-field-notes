# 03.00 - Logs

Logs are created automatically anytime you use a task and add a specific hashtag to it. 

For example:

- [ ] This is a really big risk! #risk 

**These are the hashtags being tracked**

- #issue 
- #question 
- #risk 
- #needs-improvement 
- #needs-documentation
